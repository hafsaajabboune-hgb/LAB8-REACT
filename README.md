# TP 8 : Consommer une API avec Fetch et Axios dans React
Ce projet React a été réalisé dans le cadre d'un TP sur la consommation d'API. Il compare l'utilisation de fetch() et axios pour récupérer et afficher des données.
# Fonctionnalités
Fetch() : Affichage de la liste des pays francophones avec leur capitale et population

Axios : Affichage d'un annuaire d'utilisateurs personnalisés (noms marocains, villes et entreprises)
